# Meu projeto

Meu primeiro projeto utilizando Flask, ele é um projeto que usei 
para praticar e aprender.

## Como funciona

O jogador escolhe um número entre 1 a 100.

A cada tentativa, o número muda,o jogador deve ter a sorte de
escolher o numero certo para ganha.

## tecnologias usadas

 ● Python

 ● Flask

 ● Jinja2

 ● HTML

 ● CSS

## Como executar

1. Instale o Flask(lembre de instala o python antes):

```bash
pip install flask
```

2. Entre na pasta do projeto:

```bash
cd flask_projeto
```

3. Execute o programa:

```bash
python main.py
```
